## 📌 Overview

**Nexus** is a SaaS-style frontend platform that simulates a real-world investor–entrepreneur ecosystem.

It is designed as an **advanced frontend internship project**, where the goal is not just to build UI, but to **extend an existing large-scale React codebase and evolve it into a full collaboration system**.

The application simulates real startup workflows including:
- Investor–startup communication
- Meeting scheduling
- Document sharing
- Video call interface
- Wallet & funding flow
- Role-based dashboards

> ⚠️ This project is fully frontend-based and uses local/mock data only (no backend or APIs).
